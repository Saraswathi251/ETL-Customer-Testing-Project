# ETL Testing Project

## Project Overview
This project validates customer data migrated from MySQL Source to Snowflake Target.

## Tools Used
- MySQL
- Snowflake
- SQL
- Python
- Excel
- GitHub

## Validations Performed
- Record Count Validation
- Null Validation
- Duplicate Validation
- Aggregate Validation
- Source to Target Comparison
- Transformation Validation

## Project Flow
Requirement Analysis
→ Mapping Document
→ Dataset Loading
→ MySQL Source
→ Transformation Logic
→ Snowflake Target
→ ETL Testing
→ Python Automation
→ Excel Report
